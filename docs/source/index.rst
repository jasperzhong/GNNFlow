GNNFlow is a Python package for building and training dynamic graph neural networks. 


Installation
------------

We develop the package with the following environments:

- Ubuntu 20.04LTS
- gcc 9.4
- CUDA 11.3
- cmake 3.23
- python 3.8

Python dependencies:

- `pytorch`
- `dgl`

.. code:: bash

    git clone --recursive https://github.com/jasperzhong/GNNFlow.git
    cd GNNFlow
    pip install -e .
